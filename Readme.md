Fenrir
======

Fenrir is a tool for inline modification of Git repositories. More later.

Build
-----

### Prerequisites

- [.NET Core SDK][dotnet-core-sdk] 3.1 or later

### Build

```console
$ dotnet build
```

### Run

```console
$ dotnet run --project Fenrir -- [arguments...]
```

### Test

```console
$ dotnet test
```

[dotnet-core-sdk]: https://dot.net/
