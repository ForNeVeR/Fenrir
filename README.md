Fenrir
======

Fenrir is a tool for inline modification of Git repositories. More later.

Build
-----

### Prerequisites

- [.NET SDK][dotnet-sdk] 6.0 or later

### Build

```console
$ dotnet build
```

### Run

```console
$ dotnet run --project Fenrir -- [arguments...]
```

### Test

```console
$ dotnet test
```
